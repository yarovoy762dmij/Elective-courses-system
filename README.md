# Курсовой проект: Информационная система для организации факультативов кафедрой (Вариант № 9)

Предметная область описывается следующим образом:
* Кафедра учебного заведения занимается организацией факультативов для студентов.
* О каждом студенте известны фамилия, имя, отчество, адрес и номер телефона.
* Каждая кафедра должна обеспечить проведение факультативных занятий по некоторым предметам.
* Некоторые из факультативов могут длиться более одного семестра. В каждом семестре для каждого факультатива устанавливаются объем лекций, практик и лабораторных работ в часах.
* Фиксируется информация о том, кто из студентов записался на какие факультативы.
* Есть некоторый минимальный объем факультативных предметов, которые должен прослушать каждый студент.
* По окончании семестра сохраняется информация об оценках, полученных студентами на экзаменах. В качестве итоговой оценки за предмет берется последняя оценка, полученная студентом.
